# ConfirmPopup

## APIs
| Name | Description | Type | Default |
| --- | --- | --- | --- |
| className(optional) | The extra classname for ConfirmPopup | string | - |
| prefixCls(optional) | The classname prefix for wholly control the component style | string | `'confirmpopup'` |
