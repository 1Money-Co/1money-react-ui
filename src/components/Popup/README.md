# Popup

## APIs
| Name | Description | Type | Default |
| --- | --- | --- | --- |
| className(optional) | The extra classname for Popup | string | - |
| prefixCls(optional) | The classname prefix for wholly control the component style | string | `'popup'` | 


## Example

```tsx
import { Popup } from './Popup';

<Popup />
```
