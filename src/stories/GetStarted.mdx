# @1money/react-ui

## Quick start
### Install
```shell
npm i -S @1money/react-ui
# or
yarn add @1money/react-ui
# or
pnpm i -S @1money/react-ui
```

### Usage
```js
import UI from '@1money/react-ui';
```

## Dev
For debugging or maintenance, you can clone the whole git repository and run the project:

```shell
git clone --depth 1

pnpm i && pnpm dev
```
